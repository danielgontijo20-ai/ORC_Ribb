# ORC_Ribb

Sistema para geração de orçamentos dos produtos comercializados.

## Estrutura do projeto

```
ORC_Ribb/
├── data/
│   ├── planilhas/   # Planilhas Excel de entrada (catálogo, preços)
│   └── produtos/    # Dados de produtos processados
├── docs/            # Documentação do sistema
├── src/
│   ├── components/  # Componentes da interface
│   ├── services/    # Regras de negócio e integrações
│   └── utils/       # Funções auxiliares
├── templates/       # Modelos de orçamento (PDF/HTML)
└── uploads/         # Arquivos enviados temporariamente
```
