# Web — Consulta, clone, logos e log de custos

## Orçamentos
- **Gerado / Aprovado:** somente consulta, memória, PDF e (se gerado) aprovar
- **Clonar:** disponível para todos; abre cópia editável como novo orçamento
- **Rascunho:** pode continuar edição

## Logos
- Upload em Cadastros → Valores Nativos (master / cabeçalho / rodapé)
- Menu: logo master acima dos botões, mesma largura do bloco
- Prévia da proposta usa logos de cabeçalho e rodapé

## Cadastros
- Custos editáveis com **log de alterações** (quem/quando/valor anterior/novo)
- Valores monetários exibidos com **R$**

## UX
- Botões do menu azuis com relevo, mesmo tamanho
- Ao expandir formulário/modal/NF, a tela rola até o conteúdo
